Design Research

Runtime Visual Language

9 visual metaphors for seeing what code actually does — not just what it says.

Built for Strata · Scroll to explore ↓

01 · Request Path

Request River

A river system where the main channel is the happy path, tributaries are middleware, and eddies are async ops.

Why This Metaphor

Rivers encode volume (width), speed (particle velocity), branching (tributaries), and danger (rapids) — all of which map to request behavior. Unlike subway maps, rivers show dynamics, not just topology.

Strata Integration

Each river segment maps to entities in the call graph. Blast radius = how far upstream the river goes. RuntimeEntrypoint + RuntimePath schemas provide the data backbone.

02 · Data Flow

Fluid Dynamics

A chemical process diagram where data is fluid — pipes, valves, vessels, and pressure gauges.

Why This Metaphor

Chemical process diagrams solve exactly this problem in physical engineering: tracking fluid through transformations, storage, and distribution. Color shifts at serialization boundaries make the invisible visible.

Strata Integration

DataAccess schema tracks db-read/write, cache operations. Pipe colors shift at each boundary. Agents changing a field see ALL places that field exists in different forms.

03 · Async & Events

Broadcast & Echo

Radio towers and receivers. Events are signals; listeners light up when reached. No return path.

Why This Metaphor

The fundamental problem with async events: there's no return arrow. A radio tower doesn't know its listeners. The broadcast metaphor captures fire-and-forget asymmetry perfectly.

Strata Integration

RuntimeEntrypoint already has kind: "event" | "queue" | "cron". The broadcast shows "this emit reaches THESE 7 listeners, and you changed zero of them."

04 · Hot Paths

Thermal Overlay

Infrared camera on code. Hot paths glow white, cold paths vanish into darkness.

Why This Metaphor

Thermal cameras are universally understood. Hot = important/active. Cold = dormant. Two heat maps that disagree (frequency vs latency) reveal the functions agents will optimize incorrectly.

Strata Integration

Hotspot data (complexity × churn) already exists. Layer thermal coloring on the circle-packing explorer. With production traces, switch from proxy heat to real heat.

05 · State Danger

Mutation Zones

Hazmat containment zones. State mutations are hazardous material with blast radius.

Why This Metaphor

Hazmat zones are viscerally understood: stay out unless you know what you're doing. The concentric rings communicate "closeness to danger" — code near a mutation inherits its risk.

Strata Integration

DataAccess already classifies db-write, cache-write, cache-delete. Layer danger zones on the explorer. Agents see "you're in a red zone — here's everything affected."

06 · Production Gap

Shadow Infrastructure

Ghost layer revealing hidden production infra behind innocent-looking code.

Why This Metaphor

The production gap is literally invisible in code. Feature flags, env vars, migrations, deploy configs — all haunt your code from the shadows. The slider makes the gap tangible.

Strata Integration

DataAccess tracks env and feature-flag access. The shadow layer is literally "here's everything the agent can't see." A brief says: "4 shadow dependencies: flag, migration, env var, deploy config."

07 · PII & Security

Contamination Paths

Radioactive tracking. Sensitive data leaves a glowing trail. Sanitization is decontamination.

Why This Metaphor

Radioactive contamination is the perfect analog for PII: invisible, spreads through contact, requires deliberate decontamination, leaks are catastrophic. Contamination is cumulative and sticky.

Strata Integration

Layered on RuntimePath reachable entities. Agents that add console.log(user) see the contamination warning: "this variable is radioactive — do not log."

08 · Serialization

Portal Network

Gateways between realms where data changes form. Each format is a distinct visual realm.

Why This Metaphor

Serialization is where more bugs live than anyone admits. The portal metaphor makes each boundary crossing VISIBLE. Field diff at each portal reveals data loss and type mismatches.

Strata Integration

DataAccess tracks access kinds. Agents adding fields to DTOs see "you changed Object realm but didn't update the portal to DB realm — new field will be silently dropped."

09 · Static vs Runtime

Reality Overlay

AR overlay showing runtime truth on static code. Toggle between what code says and what actually runs.

Why This Metaphor

The morph animation itself is the information. When you toggle static→runtime and watch half your code ghost away, you viscerally understand the gap between editor and production.

Strata Integration

The ultimate agent safety tool. "60% of what you're reading doesn't run in production. Here's the 40% that matters." Prevents agents from optimizing dead code.

The Gap Is The Story

Every visualization shows a gap — between code and runtime, visible and hidden, safe and dangerous. Strata makes gaps tangible so humans can steer agents through them.
